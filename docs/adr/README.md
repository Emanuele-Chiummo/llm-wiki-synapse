bounded `InferenceProvider.chat()` call (operation `"ingest"`, I6) to pre-generate 1вЂ“3 follow-up queries
(timeout + token_budget в†’ NULL on failure; **fire-and-forget, never fails the ingest** вЂ” I7/AC-F9-2); REST
`GET /review/queue` (paginated, capped) + `POST /review/queue/{id}/{approve|skip|deep-research}` (the last
delegates to `POST /research/start` and stores `run_id`). The consistency/contradiction graph scan is
**reserved + bounded** (reads `edges`, never FA2 вЂ” I2). **F12:** a sole `ingest/extract.py` with a pure
`extract_text(path) -> str` dispatch (pypdf/python-docx/python-pptx/openpyxl вЂ” static guard, **no LLM**,
I6/I9); `POST /ingest/upload` widens its accept-list and, on a binary, writes the original to
`raw/sources/` then **synchronously** extracts a companion `<stem>.extracted.md` (Obsidian-valid
frontmatter, I5) вЂ” the **watcher ingests only the companion** (binary extension stays OUT of
`_ALLOWED_EXTENSIONS`), so one `ingest_file`, one hash gate (I1), one wiki page; images/AV are
**placeholder-only** and **`unstructured` is deliberately deferred** to M6 (lean deps, I9). D2 adds
`review_items`; D4 adds the 4 review endpoints + widened upload. ADR-0026 closes sprint v0.5 (M5
Phase 4) locking **F13 cascade deletion** вЂ” the highest vault-integrity risk in M5 вЂ” before engineers
code: a **single-pass, inference-free, FA2-free** `cascade_delete(page_id)` in `ops/cascade_delete.py`.
References to a deleted page are found by a **3-method match** вЂ” (a) exact `links.target_page_id`/
`target_title`, (b) slug-normalised `links.target_title`, (c) a **bounded, last-resort** fulltext
fallback over live `wiki/` вЂ” where the primary methods read the **`links` back-reference index** (built
incrementally by `persist_links`) and **NEVER rescan the vault** (I1; the (c) fallback is scoped to the
enumerated live-wiki set and skipped on the links-hit happy path). The **preserve-shared rule** is
exact: delete a page **iff** its frontmatter `sources[]` becomes **empty** after removing the source,
else **preserve + prune** the single `sources[]` entry (shared pages are never deleted; source-overlap
`edges` pages are **WARNED, never blocked** вЂ” AC-F13-3). Dead `[[wikilinks]]` are rewritten to plain
text (`[[T|alias]]`в†’`alias`) in the **body only** of **only** the referencing files via the
`frontmatter.loads`/`dumps` round-trip (never editing inside the `---` fences вЂ” **I5**;
`test_obsidian_check.py` 15/15 is the gate). The data plane mirrors `delete_file` (soft-delete
`deleted_at`, ADR-0005; `delete_point()`), regenerates `index.md` from live pages (K3 entry auto-removed),
deletes the originating `raw/sources/` file (**AQ-v0.5-5** вЂ” no restart resurrection), and bumps
`data_version` **EXACTLY ONCE** so the **debounced** GraphCache recompute fires (**no inline FA2** вЂ” I2).
F13 is **migration-free** (reuses `pages.deleted_at`, `links`, the two `edges` endpoint indexes,
`pages.sources`) вЂ” **no D2 change**. A **mandatory dry-run** separates plan from apply: read-only
`plan_cascade_delete()` / `POST /pages/{id}/cascade-delete/preview` (200, mutates nothing) returns
`{will_delete, will_preserve_with_pruned_source, wikilinks_to_rewrite, shared_entity_warnings,
match_methods_used}`; the destructive apply is `DELETE /pages/{id}` (200; 404 on a double-delete).
D3 adds `cascade-delete.mmd`; D4 adds both endpoints.
