# Synapse v1.2.2 — Home Fixes and Type Reclassification

_Released: 2026-07-03_

Patch fixing a Home dashboard crash and the full-width layout constraint, and adding bounded AI-driven page-type reclassification against the curated `schema.md`.

## Added

- **Bounded page-type reclassification** (`POST /ops/reclassify-types`) — an AI pass that automatically corrects AI-assigned page types against the vault's curated `schema.md` rules; single-flight, bounded by `max_iter` + `token_budget`, cost logged (I7). Implements K8: the human curates the rules, the model applies them [F18, K8].

## Fixed

- **Home dashboard full-width** — removed the 1,100 px cap that constrained the layout on wide screens [F18].
- **Backfill summary crash** — a domain-classification section summary serialised as an object (not a string) caused a React render error on the Home dashboard [F18].
