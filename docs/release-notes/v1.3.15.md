# Synapse v1.3.15 — Marker GPU Fix for RTX 3060

_Released: 2026-07-09_

Single-fix patch: pins PyTorch to the `cu128` index in the Marker Dockerfile so GPU conversion works correctly on TrueNAS hosts whose NVIDIA driver caps out at CUDA 12.8.

## Fixed

- **Marker GPU support on RTX 3060** — the default PyPI `torch` wheel targets a CUDA version newer than the TrueNAS host driver supports (cap: CUDA 12.8), causing `TORCH_DEVICE=cuda` to fail at runtime with "NVIDIA driver on your system is too old" (HTTP 500 from `/convert`). Pinning `torch==2.7.1` and `torchvision==0.22.1` from the PyTorch `cu128` index before `marker-pdf` in the Dockerfile resolves the mismatch [F12].
