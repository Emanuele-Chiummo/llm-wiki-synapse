# Synapse v1.2.0 — Home & Insights

_Released: 2026-07-03_

The Home dashboard becomes Synapse's central landing page, showing live system health across seven components, active jobs, wiki KPIs, auto-clustered knowledge groups, and optional curated domain sections; the backend now publishes to GHCR on every release and supports zero-touch Watchtower updates. Verified with 1,665 backend and 1,632 frontend tests; ER/OpenAPI zero-drift (36 endpoints); live-tested on the real vault.

## Added

- **Home dashboard** — a new landing page replacing the direct wiki view: live health of seven components (DB, Qdrant, watcher, queue, graph cache, embeddings, scheduler), active provider, version, uptime; active-jobs panel (ingest, Deep Research, domain classification disappear when idle); wiki KPIs (pages, links, communities, pending reviews, lint issues, monthly AI spend); recent activity with direct page navigation [F18].
- **Automatic knowledge groups** — graph community clusters surface as grouped cards with counts, composition, and lead page; first four shown with expand; zero configuration [F4, F18].
- **Curated domain sections** — define a controlled domain vocabulary (e.g. ServiceNow, SAM, TPRM) and the AI classifies every page; new pages classified at ingest, existing pages via a bounded backfill [F18].
- **Job scheduling** — lint scan and domain classification are schedulable (off / hourly / daily / weekly) from Settings → Advanced with a manual "Run now" and last-run outcome. Lint fixes remain human-approved [K2, F18].
- **GHCR images on every release** — `backend` and `marker` tagged with the release version and `latest` [F16].
- **Optional Watchtower** (`--profile autoupdate`) — zero-touch backend-only updates from GitHub; databases are never touched [F16].
- **Marker native in Docker Compose** (`--profile marker`) — CPU path on the server or CUDA on TrueNAS (RTX 3060); model weights stored in a persistent volume [F12].
- **Desktop "Launch Marker" button** — if the Marker microservice is stopped, the desktop app can start it (configurable command, polls until ready) [F12, F15].
- **`vault/` fully excluded from the repository** — the backend recreates the full directory structure on first run [K1].
- **Version comparison banner** — the app compares its own version with the server's and displays an advisory when the server is behind [F15].
