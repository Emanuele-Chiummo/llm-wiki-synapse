# Synapse 1.3.4 — Allineamento UI a llm_wiki 🪞

Release grande: un intero programma di parità **screenshot-driven** con nashsu/llm_wiki, feature per feature, mantenendo gli interni superiori di Synapse (grafo server-side, indice incrementale, loop bounded). Tutto verificato dal vivo contro il vault reale. Governato da ADR-0058 → ADR-0061.

## 🧹 Lint — parità piena
- Nuova categoria deterministica **Link interrotto** (broken-wikilink) dai dangling link della tabella `links` — a costo LLM zero (era tracciata ma mai mostrata)
- **Suggested target** (risolutore tollerante) + **Correggi** che riscrive il link nel body
- **Batch bar** (Correggi/Ignora/Invia a revisione selezionati), **Apri** per riga, **bridge → coda Revisione**, header per severità con **totale vero** (es. "Avvertimenti (259)"), toggle **Semantico (LLM)**, **Elimina** per pagine orfane

## 💬 Chat — composer come llm_wiki
- **Allega immagine** (capability-aware sul provider, I6), toggle **Ricerca web** (SearXNG, namespace citazioni `[W]` separato), modalità **Veloce / Standard / Profondo / Locale prima** (preset deterministici). AnyTXT ⛔ non-applicabile (servizio solo-Windows)

## 🕸️ Grafo — il grosso del lavoro
- **Header**: statistiche `pagine/link/nascosti`, **Cerca** nel grafo, **Filtro** per tipo (I2-safe), Reset, schermo intero
- **Community per dominio** con **nomi unici** ("SAM · Reconciliation", "SAM · Reporting") — niente più duplicati; colorate per cluster Louvain, nominate dal dominio F18
- **Flusso "Obsidian"**: culling degli archi ridotto → cluster riconnessi (non più isole); il chip link riflette il culling
- **Legenda collassabile**; pannello Insights **collassato di default**; etichette centroide clampate al canvas
- Le **dimensioni** dei nodi restano scalate per link count (√)

## 📁 Sorgenti & Lettore
- **Import cartella** ("+ Folder", ricorsivo) + **elimina cartella** (cascade bounded) + footer conteggio
- Reader: **"More" sui tag** (i ~100 tag non wrappano più all'infinito) + riga `updated:`
- **Nuovo tab "Wiki"** nelle Sorgenti: sfoglia la struttura della cartella `wiki/` (sola lettura, con preview) — come il "Files" di llm_wiki

## 🔌 MCP + Agent Skill · 🔬 Deep Research
- MCP esteso da 4 a **9 tool** (graph-neighborhood, list/resolve-reviews, read-source-file, trigger-rescan) + REST `review/bulk-resolve` + `PATCH`
- **Skill installabile** `tools/synapse-skill/` — interroga Synapse da Claude Code/Codex (read-only, trigger-disciplined)
- **Deep Research**: dai Graph Insights, topic + query **ottimizzati dall'LLM** (legge overview+purpose) in un **dialog editabile** prima di partire
