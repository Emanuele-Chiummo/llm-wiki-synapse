# Synapse 1.2.6 — ingest bulk senza OOM 🛡️

Release singola con un fix di robustezza importante per l'uso reale.

## Fix
- **Concorrenza dell'ingest limitata (ADR-0056)**: trascinare decine di file in `raw/sources/` lanciava altrettanti ingest simultanei — pool DB esaurito, GPU sommersa di richieste embedding, picco RAM fino al crash dell'host. Ora un semaforo (`INGEST_MAX_CONCURRENCY`, default 3) processa tutto in coda ordinata: cambia il *quando*, mai il *se* (I7)
