# Synapse v1.2.1 — Graph Dark Mode Fix and Proactive Updates

_Released: 2026-07-03_

Patch above v1.2.0 fixing invisible graph edges in dark mode and a faulty server-behind version comparison, and adding periodic update checks every four hours.

## Added

- **Periodic update checks** — the app checks for new releases every 4 hours and on window-focus return; the update banner appears automatically without restarting [F15].

## Fixed

- **Graph edges visible in dark mode** — edge colour ramps now follow the active theme; the previous dark background rendered all edges invisible [F4].
- **Version banner fires only when the server is actually behind** — the comparison is now correct semver (`server < app`), not a string equality check that could trigger on an already-updated server [F15].
