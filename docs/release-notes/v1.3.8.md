# Synapse v1.3.8 — App iOS nativa

> Aggiunge il primo client **iOS nativo** (SwiftUI). Nessuna modifica al backend,
> nessuna migrazione DB. Le immagini desktop/container sono ribuildate solo per
> allineamento di versione.

## ✨ Novità

- **App iOS nativa (SwiftUI, iOS 17+)** `[F15]` — nuovo client in `ios/` che si
  connette al backend Synapse via REST. Ricrea il design mobile del handoff
  Claude Design con 5 tab:
  - **Wiki** — statistiche del vault, pagine recenti e complete, dettaglio con
    corpo Markdown, collegamenti e mini-grafo locale
  - **Cerca** — ricerca RAG a 4 fasi con filtri per tipo
  - **Chat** — risposte in streaming (NDJSON) con citazioni tappabili che aprono
    la pagina di origine
  - **Grafo** — knowledge graph interattivo con pan/zoom in stile Obsidian ed
    etichette hub anti-sovrapposizione
  - **Altro** — coda di revisione, import, deep research, impostazioni
    (URL server + token, aspetto, provider)
  - Progetto XcodeGen (`ios/project.yml`), firma automatica, guida all'auto-firma
    in [`ios/README.md`](https://github.com/Emanuele-Chiummo/llm-wiki-synapse/blob/main/ios/README.md).

## 📎 Artifact

- **`Synapse-1.3.8-unsigned.ipa`** — build iOS non firmata, allegata alla release.
  Va firmata col tuo Apple ID (Xcode / [AltStore](https://altstore.io) /
  [Sideloadly](https://sideloadly.io)); con account gratuito l'app è valida 7 giorni.
- Bundle desktop macOS/Windows + immagini GHCR come di consueto.

## ⬆️ Aggiornamento

Backend e frontend invariati a livello funzionale rispetto a v1.3.7 — l'update
container è opzionale (solo allineamento versione). Nessuna migrazione DB.
