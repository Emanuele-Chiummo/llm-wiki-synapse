# Synapse v1.3.8 — Native iOS App

_Released: 2026-07-06_

Introduces the first native iOS client — a SwiftUI app targeting iOS 17+ with five tabs covering the full Synapse feature set via REST. The backend and container images are rebuilt for version alignment only; no backend changes, no DB migration. An unsigned IPA build is available for self-signing using `ios/build-unsigned-ipa.sh`.

## Added

- **Native iOS app (SwiftUI, iOS 17+)** in `ios/` [F15]: connects to the Synapse backend via REST and provides five tabs — Wiki (vault stats, page list, detail with Markdown body and mini-graph), Search (4-phase RAG with type filters), Chat (NDJSON streaming with tappable citations that open the source page), Graph (interactive pan/zoom with hub-label anti-overlap), and Altro (review queue, import, deep research, settings including server URL + token, appearance, and provider). XcodeGen project at `ios/project.yml`; setup guide at [`ios/README.md`](https://github.com/Emanuele-Chiummo/llm-wiki-synapse/blob/main/ios/README.md).

## Upgrade notes

- Backend and frontend are functionally unchanged from v1.3.7; container update is optional (version alignment only). No DB migration.
- An unsigned IPA build can be reproduced locally using [`ios/build-unsigned-ipa.sh`](https://github.com/Emanuele-Chiummo/llm-wiki-synapse/blob/main/ios/build-unsigned-ipa.sh) and self-signed with your Apple ID via Xcode, AltStore, or Sideloadly. With a free Apple account the app is valid for 7 days.
