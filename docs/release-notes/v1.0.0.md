# Synapse v1.0.0 — Distribution

_Released: 2026-07-03_

First distributed release: signed desktop bundles, auto-update from GitHub Releases, shared-token authentication, mobile/PWA layout, and a published MkDocs documentation site. Users on v0.8.1 or later received this update automatically via the in-app banner. 1,563 backend + 1,430 frontend + 35 E2E tests green; invariants I1–I9 verified.

## Added

- **Shared-token authentication**: `SYNAPSE_AUTH_TOKEN` environment variable protects all backend routes with constant-time comparison; empty = disabled for full backward compatibility. The desktop app prompts for the token on connection; the web UI shows a gate screen; MCP and the web clipper retain their separate dedicated tokens. Configurable from Settings → Security.
- **Mobile and PWA layout (F15)**: below 768 px the rail collapses to icon-only; panels stack vertically; touch targets are ≥ 44 px; pinch-zoom on the graph is corrected. Installable as a PWA on iOS and Android.
- **Documentation site**: MkDocs Material site published at `https://emanuele-chiummo.github.io/llm-wiki-synapse/` — user guide, deploy guide, C4 architecture, 52 ADRs, and API reference; deployed automatically from `main`.
- **Code-signing guide**: step-by-step instructions for Apple Developer and Windows certificate setup, CI secrets, and the signing workflow documented in `DEPLOY.md §14`; when certificates are in place, Gatekeeper/SmartScreen prompts disappear and the experience is indistinguishable from a store-distributed app.
