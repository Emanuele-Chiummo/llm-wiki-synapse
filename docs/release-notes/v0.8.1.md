# Synapse v0.8.1 — hotfix auto-update

_Released: 2026-07-03_

Critical patch: the in-app auto-update banner never appeared in v0.7.0 or v0.8.0 because the Tauri updater plugin was excluded from the JS bundle at build time. Install this release manually once — all subsequent updates will arrive automatically via the in-app banner from this version onward. Content is otherwise identical to v0.8.0.

## Fixed

- **Auto-update plugin missing from JS bundle**: dynamic imports of the Tauri updater plugin were not resolvable at build time, so the update-check failed silently and the banner never appeared in v0.7.0 and v0.8.0. The plugin is now correctly included in the bundle.

## Upgrade notes

- Install v0.8.1 manually one time (download the `.dmg` or `.exe` below). After this one-time step, all future updates will be delivered automatically via the in-app banner.
