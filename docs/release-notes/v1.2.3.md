# Synapse v1.2.3 — In-App Reclassification and Exact Badges

_Released: 2026-07-03_

This patch promotes page-type reclassification to a proper fourth scheduled operation with a UI trigger, adds an always-visible pending-review badge with an exact count (no more "9+"), and introduces a weekly schema-review job. Verified with 1,704 backend and 1,642 frontend tests; full CI gates green.

## Added

- **In-app type reclassification as a scheduled operation** — a fourth entry under Settings → Advanced → Scheduled jobs; configurable frequency + "Run now". Reclassifies new, untyped, or mis-typed pages according to `schema.md`; already-correct pages are skipped [F18, F9].
- **Pending-review badge on the Review rail item** — always-visible count of items awaiting action, sourced from the existing `/status` poll [F9, F1].
- **Weekly schema-review operation** — a schedulable job proposes `schema.md` conformance changes via the Review queue (K6 co-evolution; human approval always required) [F18, K6, K8].
- **Exact badge counts** — the review pill now expands to display the real number (ceiling 999+) instead of truncating at "9+" [F1].
