# Synapse v1.5.3 — Synthesize/comparison UI trigger

_Released: 2026-07-11_

Patch: exposes the corpus-level synthesis/comparison generator in the Home dashboard — previously API-only with no UI trigger. The synthesis run remains a manual, on-demand operation; auto-triggering on bulk-ingest completion is tracked as a follow-up.

## Added

- **"Generate now" nudge for synthesis/comparison pages** — a new Home dashboard banner triggers the bounded corpus-level synthesis/comparison generator on demand. High-confidence clusters are auto-written as synthesis or comparison pages; borderline clusters are proposed to the Review queue. Hidden when the corpus has fewer than 3 entity/concept pages or while a run is already in flight. A "Syntheses/comparisons in progress" indicator appears in active-jobs while the run is live [F18, ADR-0067 D3].
