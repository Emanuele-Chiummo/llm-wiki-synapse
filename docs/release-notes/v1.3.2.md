# Synapse v1.3.2 — Live QA Pass: Four Targeted Fixes

_Released: 2026-07-05_

A live QA pass against the production instance surfaced four targeted fixes in the setup wizard, Deep Research, Search, and the Review queue; no structural changes. Verified with 1,761 frontend tests and 16 Deep Research backend tests green; fix scenarios verified visually against production data.

## Fixed

- **Setup wizard: backend server URL is now editable** — the Step 1 URL field displayed as read-only text; it is now an editable field with `http`/`https` scheme validation, a live probe on the entered URL, and persistence via `setServerUrl`. Leaving it empty uses the page's own origin [F16].
- **Deep Research: zero-source runs no longer produce a junk page** — when SearXNG is unreachable or returns no results, the synthesis step and subsequent ingest are now skipped (no wiki page created) [F10, I7].
- **Search: relevance percentage shown only for vector results** — graph-expansion results were displaying raw graph weights as percentages (e.g. 2144%); the `%` chip now appears only for vector similarity scores (0–100 %, clamped); expansion results show only the badge [F5].
- **Review queue: auto-resolve button relabelled** — two buttons labelled "Clear resolved" appeared in non-pending tabs; the auto-resolve action is now labelled "Auto-resolve" to distinguish it from clearing terminal rows [F9].
