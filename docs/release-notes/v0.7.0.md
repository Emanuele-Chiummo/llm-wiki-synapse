# Synapse v0.7.0 — Core completeness & daily UX

_Released: 2026-07-03_

First release with a working in-app auto-update channel: `latest.json` is minisign-signed and verified before installation. All future updates from v0.7.0 onward arrive automatically via the in-app banner — no manual download needed. 1,373 backend and 1,277 frontend tests green; ADRs 0049 (auto-update) and 0050 (retrieval scope) committed.

## Added

- **In-app auto-update (ADR-0049)**: at launch the desktop app checks GitHub Releases and shows an "Update now / Later" banner; updates are minisign-signed and verified before install. This release ships the first signed `latest.json`.
- **New wiki page from the tree**: create a page (title, type, directory) directly from the wiki tree without leaving the app.
- **Rename and search conversations**: chat conversations can be renamed and searched by title.
- **Unsaved-change indicator**: the editor flags unsaved changes and prompts before navigating away.
- **Bulk operations on sources**: multi-select sources; ingest or delete them in a batch with a progress indicator.
- **Vault scenario presets**: 5 preset vault templates (Research, Reading, Personal Growth, Business, General) applicable from Settings.
- **Streaming reasoning preview**: the `<think>` block streams live during generation; a confirmation dialog guards "Cancel all".
- **Review-generated search queries**: review items carry pre-generated queries reused as seeds for Deep Research.
- **Recursive folder import**: `IMPORT_SCAN_RECURSIVE` flag; folder path injected as analysis context.
- **ServiceNow connector daemon**: watched folder → Marker conversion → auto-ingest; launchd template included.

## Fixed

- **DeepSeek/Qwen/Ollama reasoning routing**: reasoning-provider detection corrected for models that embed `<think>` inside the response body.
- **Mandatory output-language directive**: all three provider backends now receive an explicit language directive so generated pages match the vault language.
- **Synthesis page classification**: synthesis pages are correctly typed instead of landing in the wrong category.
- **Citations source scope (ADR-0050)**: chat citations reference only `wiki/` pages, never raw source files.
