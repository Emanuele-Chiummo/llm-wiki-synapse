# Synapse v0.8.0 — Content power

_Released: 2026-07-03_

First release delivered via in-app auto-update to v0.7.0 installs. Adds high-quality PDF extraction via Marker (opt-in), AI image captions, audio/video transcription, filtered search, clickable citations in chat, and streaming vault backup. A formal UX audit was conducted and documented; fixes begin in v0.9. 1,446 backend and 1,308 frontend tests green.

## Added

- **High-quality PDF extraction via Marker (F12, ADR-0051)**: opt-in `PDF_EXTRACTOR=marker`; Marker runs as a microservice on the host with GPU/MPS acceleration, preserving tables and document structure; pypdf remains the always-available fallback.
- **AI image captions (opt-in)**: PNG/JPG files in the vault are described by a vision-capable provider and enter the knowledge graph; captions are cached by file hash; capped at 5 per ingest run.
- **Audio/video transcription (opt-in)**: local Whisper service (Apple Silicon MPS or CPU) transcribes audio/video files; capped at 3 per run.
- **Search facets and sorting**: filter results by page type; sort by relevance or date.
- **Clickable citations in chat**: clicking `[n]` opens the cited wiki page (previously a stub with no navigation).
- **Vault backup**: `GET /export` streams a ZIP of the vault (≤ 500 MB); `GET /export/data.json` dumps pages, links, graph data, and run history. Restore procedure documented in `DEPLOY.md`.
- **Chrome web clipper on every release (F11)**: `synapse-clipper-0.8.0.zip` attached with an unpacked-installation guide; permissions and brand icons corrected.

## Changed

- **Citations source scope (ADR-0050)**: raw source files never appear as citation targets — citations come exclusively from `wiki/`.
