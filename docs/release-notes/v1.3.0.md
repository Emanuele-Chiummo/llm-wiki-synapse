# Synapse v1.3.0 — Foundations

_Released: 2026-07-05_

The sprint that pays down structural debt before multi-vault work: `main.py` is decomposed from 9,311 into ~1,400 lines across 13 domain routers, responsive mobile/tablet/desktop layouts land (ADR-0057), and two P1 + 18 P2 correctness issues are resolved with no new AI features by design. First release cut from `main` under the new tagging policy. Verified with 1,820 backend and 1,750 frontend tests; mypy strict 0 errors; OpenAPI zero-drift; ADRs 0056 and 0057 indexed.

## Added

- **Responsive layouts (ADR-0057)** — three viewport tiers (mobile ≤767 px / tablet 768–1023 px / desktop ≥1024 px): drawers for the page tree, detail panel, and conversations on mobile; safe-area insets for notch/home-indicator; `100dvh` for Safari. No ~350 ms tap delay, ≥44 px touch targets, instant tap feedback, conversation actions always reachable on touch [F15, F16].
- **Multi-arch Docker images (amd64+arm64)** — Synapse now runs on Raspberry Pi / ARM servers as well as TrueNAS (amd64) [F15].
- **E2E Playwright job in CI** — seeded Docker stack (340-node graph, providers, conversations, sources on disk) with the correct API base and vault paths; 122+ tests run on every push to `main`; hardware-aware skips for tests requiring live Ollama (`E2E_LIVE_CHAT=1`) or a real GPU (`E2E_SOFTWARE_GL`) [F15].
- **`make bump`** — atomic version bump across four files [F15].

## Changed

- **`main.py` decomposed** — 9,311 → ~1,400 lines across 13 domain routers (`app/routers/*`); API contract frozen and proven byte-identical OpenAPI before and after.
- **Release lineage realigned** — v1.2.4–1.2.6 merged into `main`; "tags are cut only from main" rule documented in `CONTRIBUTING`.

## Fixed

- **Graph recompute no longer freezes the server** — igraph/FA2/Louvain layout now runs in a thread executor; requests, streams, and the watcher remain fluid even on vaults with thousands of pages (I2) [F4].
- **Chat responses bound to their originating conversation** — stream aborts on conversation switch or unmount; responses no longer arrive in the wrong conversation [F6].
- **Atomic `index.md` writes**, provider streams closed on timeout/error, word-boundary wikilinks (`[[Cat|cat]]egory` no longer mismatches), concurrent-edit **409** instead of silent data loss, and ~14 other regression-tested fixes (2 P1 + 18 P2 from a code review of v1.2.6).

## Security

- **SSRF guard on Deep Research fetches** — only `http`/`https` schemes, private/metadata IP blocking on every redirect hop, maximum 3 redirects [F10].
- **Per-method auth exemptions** — auth is now exempted by method as well as by path; Postgres is no longer host-exposed; credentials flow via `.env`; per-IP rate limiting on chat, ingest, and research start [F17].
