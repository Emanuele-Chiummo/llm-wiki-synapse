# Synapse v0.4.0 — Usable & fluid

_Released: 2026-06-29_

Milestone M4: the 3-panel web UI is live and usable. Chat streams tokens end-to-end with NDJSON, the graph viewer renders in WebGL via sigma.js, and the provider selector UI (F17) is wired to the inference-provider table. 460 backend and 370 frontend tests green; invariants I1–I9 respected.

## Added

- **3-panel shell (F1)**: resizable, collapsible tree / chat / preview panels with a labeled nav rail (Chat · Wiki · Sources · Graph · Settings).
- **Streaming chat (F6/F7/F8)**: multi-conversation persistent chat with NDJSON streaming; `<think>` reasoning blocks stream live and collapse by default; LaTeX converts to Unicode at stream end — not per-token (I3).
- **Knowledge graph viewer (F4)**: 4-signal edge weights, FA2 server-side layout (I2), sigma.js WebGL renderer with hover-dim, node sizing, and drag-to-pin.
- **Provider selector UI (F17)**: Settings → LLM Models — add/delete Local (Ollama), API (Anthropic / OpenAI-compatible), and CLI providers; `base_url` field for Ollama; model ID presets (`qwen2.5:3b` for local, `claude-sonnet-4-6` for API).
- **Document upload**: drag-drop upload in Sources (`.md` / `.txt` / `.markdown`, 25 MB cap).
- **Scheduled folder import**: configurable import frequency with a run-now trigger.
- **Embeddings display in Settings**: live read-only view of `EMBEDDING_URL` / `EMBEDDING_MODEL` / `EMBEDDING_DIM` to confirm bge-m3 is wired without opening `.env`.
- **`docker-compose.dev.yml` override**: full local stack (Postgres + Qdrant + FastAPI backend + Vite HMR) with a single `docker compose` command.
