# Synapse v1.3.1 — Native Multi-Arch Frontend and E2E CI

_Released: 2026-07-05_

Release-engineering patch above v1.3.0: the multi-arch frontend image now builds natively in minutes rather than 4+ hours under QEMU emulation, and the CI E2E job is green for the first time with 122+ Playwright tests running against a seeded live stack on every push to `main`.

## Changed

- **Multi-arch frontend image built natively** — the Vite bundle (architecture-independent) now builds with `--platform=$BUILDPLATFORM`; only the nginx stage remains per-arch. The arm64 build in v1.3.0 was stuck for over 4 hours under QEMU emulation [F15].
- **CI E2E job green for the first time** — seeded stack (340-node graph, providers, conversations, sources on disk), bundle built with correct API base and vault paths, Playwright suite updated to the v1.3 UI — 122+ tests against the live stack on every push to `main`; hardware-aware skips for Ollama (`E2E_LIVE_CHAT=1`) and GPU (`E2E_SOFTWARE_GL`) tests [F15].
- **`release-cut` and `release-notes-sync` workflows** — tag + release from GitHub Actions; release notes versioned in `docs/release-notes/`. Release notes added retroactively for v1.2.1, v1.2.4, v1.2.5, and v1.2.6.
