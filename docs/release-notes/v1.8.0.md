# Synapse v1.8.0 — "one-click system update"

## Added

- **One-click system update (Settings → Info).** When a newer GitHub Release exists, Settings →
  Info shows "Update available: vX.Y.Z" and an **Update system** button that pokes Watchtower's
  HTTP API (`POST /ops/system-update`) to pull the latest images and recreate the labelled
  containers — no more manual `docker pull`. Availability comes from `GET /ops/update-status`
  (running version vs the latest GitHub Release, cached ~1h). Requires Watchtower's HTTP API and
  a shared `WATCHTOWER_HTTP_API_TOKEN` (the compose now wires it for backend + marker); when the
  token is unset the button is hidden and the availability line still shows. See `docs/DEPLOY.md`
  §9.2b.

## Upgrade

No Alembic migrations. Pull new images and ensure `WATCHTOWER_HTTP_API_TOKEN` is set in your
`.env` if you want the one-click update button to appear.
