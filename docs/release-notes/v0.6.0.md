# Synapse v0.6.0 — Shippable

_Released: 2026-07-03_

Milestone M6: Synapse ships as a desktop application. Tauri v2 wraps the web UI into macOS and Windows installers, a new brand identity lands, dark mode and the command palette arrive, and the bounded lint-fix loop and the Chrome web clipper close the sprint. Desktop binaries at this milestone are unsigned (right-click → Open on macOS; SmartScreen bypass on Windows). 1,191 frontend and 1,330 backend tests green; 48 ADRs committed.

## Added

- **Tauri v2 desktop app (F15)**: macOS Apple Silicon `.dmg` and Windows `x64-setup.exe` installers; auto-detects a local backend at first launch; remembers previously connected servers.
- **Brand identity**: new logo (synaptic S over a knowledge-graph motif), complete icon set, favicon, and PWA manifest.
- **Dark mode (F16)**: System / Light / Dark theme picker.
- **Command palette (F16)**: `⌘K` searches vault sections and wiki pages; keyboard shortcuts `⌘N`, `⌘1`–`⌘5`.
- **Bounded lint-fix loop (K2)**: lint proposals require human approval before applying fixes.
- **Ingest queue controls**: per-job cancel, pause, and retry.
- **Chrome web clipper (F11)**: Readability + Turndown → local API → auto-ingest with project picker.
- **Discursive `overview.md` (F3)**: llm_wiki-style narrative overview generated after ingest.
- **Deterministic FA2 layout with Louvain communities**: outlier coordinate clamp and community colouring applied at layout time.
- **MkDocs Material documentation site (optional)**: `mkdocs.yml` pointing at `docs/` with Mermaid plugin enabled.

## Changed

- **Multi-server support**: the desktop app can connect to multiple Synapse backends.
- **System notifications**: desktop notifications fire at the end of an ingest run (desktop builds only).
