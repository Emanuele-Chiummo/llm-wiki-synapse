# Synapse v1.3.13 — llm_wiki 0.6.0 Parity: Ingest, Graph, Lint, and iOS Refresh

_Released: 2026-07-09_

Brings ingest pipeline, graph rendering, and lint checks to 1:1 functional parity with `nashsu/llm_wiki` 0.6.0 (ADR-0063, ADR-0058 §7, ADR-0016 amended, ADR-0045 amended), adds a multi-page fan-out path to the review queue (ADR-0064), delivers an iOS neural visual refresh, and fixes four regressions including iOS chat streaming and an inverted review duplicate-sweep rule. No DB migration.

## Added

- **Ingest parity with llm_wiki 0.6.0** (ADR-0063) [F3][F12]: long-source chunking with checkpoint analysis (`ingest_long_source_char_threshold`); LLM body-merge on re-ingest (`ingest_reingest_merge_enabled`); deterministic wrong-language page drop (`ingest_language_guard_enabled`); generation now receives the full source document text budget-trimmed to `ingest_generation_source_char_budget` (default 24 000 chars) instead of the lossy Analysis JSON — the primary cause of wiki/graph divergence from llm_wiki on identical raw + model; `GENERATE_SYSTEM` prompt gains an explicit page-type scaffold (source-summary + entity + concept per file; synthesis/comparison reserved for the review queue); `_ensure_source_summary` guarantees exactly one origin source page per file; `index.md`/`log.md` persisted as `Page` rows (types `index`/`log`) so they appear as graph nodes; `files` multipart field name corrected (was `files[]`).
- **Graph parity with llm_wiki 0.6.0** [F4]: edges from resolved wikilinks only — shared-source contributes to edge weight, not topology; FA2 layout parameters matched (`outboundAttractionDistribution=False`, `jitterTolerance=1+ln(n)`, `barnesHutTheta=0.5`, outlier clamp removed); Tailwind-400 node-type palette (entity #60a5fa, concept #c084fc, source #fb923c, synthesis #f87171, comparison #2dd4bf, query #4ade80, overview #facc15); neutral slate edge ramp; graph chrome unified to a single top toolbar; query-type nodes excluded; hover-only labels with neighbour highlight; `hideLabelsOnMove`/`hideEdgesOnMove` enabled.
- **Lint parity with llm_wiki 0.6.0** (ADR-0058 §7) [K2]: new deterministic `no-outlinks` (info severity) and semantic `suggestion` categories; fuzzy token-overlap suggestions for orphan/no-outlinks; auto-apply paths (append `[[link]]` under `## Related` for no-outlinks/orphan; create a `type:query` stub for unresolved broken-wikilinks); `overview.md` now eligible for orphan/no-outlinks checks; client-side chunking of batch lint actions (200-id server cap); missing-xref semantic type removed.
- **Review queue multi-page fan-out** (ADR-0064) [F9]: a missing-page item whose title encodes a comma-, CJK-, or "and"/"e"-delimited list now creates one page per candidate on "Create", capped at 5 (I7).
- **iOS neural visual refresh** [F15][F16]: indigo→violet gradient for wordmark and primary CTAs; per-type SF Symbol glyphs in list rows; expressive stat cards with icon chips; pulsing gradient on the pending-review card when count > 0; `NeuralMotif` constellation behind the Wiki hero; `AuroraBackground` (drifting blurred type-colour blobs, honours Reduce Motion) behind the graph; soft elevation on hero cards.

## Fixed

- **iOS chat streaming** — the iOS NDJSON parser expected `{token, done, error}` keys but the backend emits `{"type","delta"}` events; `SynapseClient` stream decoding rewritten to match — tokens and citations now render [F6].
- **iOS sub-page header alignment** — `BackHeader`'s `VStack` pinned to full width and left-aligned, fixing offset appearance on Settings, Deep Research, Review, and Ingest screens [F1].
- **Lint orphan under-report** — inbound-link count was including links from `index.md`, `log.md`, and other vaults; inbound set now restricted to this vault's `wiki/%` content pages, excluding index/log [K2].
- **Review duplicate-sweep rule inverted** — auto-resolved duplicates when the page existed (should resolve only when an affected page is gone, llm_wiki parity); missing-page items now resolve when the page exists by slug; `_normalize_title` strips "Missing page:"/"Duplicate page:" prefixes [F9].
