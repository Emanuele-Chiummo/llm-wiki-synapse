# Synapse v1.6.1 — Home navigation cancellation fix

v1.6.1 is a patch release for a desktop/webview navigation regression in the Home dashboard.

## Fixed

- Returning to Home after navigating across sections could show
  **"The knowledge overview could not be loaded"** / **"Impossibile caricare la panoramica della conoscenza"**
  with technical detail `Request cancelled`.
- The Home stats loader now treats Tauri/native HTTP `Error("Request cancelled")` the same way as
  browser `AbortError`: a normal navigation/refetch cleanup, not a user-visible backend failure.

## Impact

- No backend, database or vault migration is required.
- Existing v1.6.0 generation parity, review, synthesis and desktop packaging behavior is unchanged.
- The fix is limited to the Home dashboard's stats loading path.
