# Synapse v1.6.1 — Home navigation cancellation fix

_Released: 2026-07-13_

Patch: returning to Home after navigating across sections could show "The knowledge overview could not be loaded" with technical detail `Request cancelled`. Frontend test suite: 156 passed.

## Fixed

- **Home stats loader crashed with "Request cancelled" on navigation** — returning to Home triggered a visible error state instead of a silent refetch. The loader now treats Tauri/native HTTP `Error("Request cancelled")` the same as browser `AbortError`: a normal navigation/refetch cleanup, not a user-visible backend failure.

## Upgrade notes

- **No migration required** — no backend, database, or vault change. Existing v1.6.0 generation parity, review, synthesis, and desktop packaging behavior is unchanged.
