# Synapse v0.9.0 — Trust & observability

_Released: 2026-07-03_

Observability, cost controls, and wiki self-improvement features land ahead of v1.0. Users on v0.8.1 or later received this update automatically via the in-app banner. 1,522 backend + 1,389 frontend + 27 E2E Playwright tests green on the real vault stack; invariants I1–I9 verified.

## Added

- **Cost dashboard**: Settings → Costs shows monthly AI spend aggregated by operation, provider, and day; configurable alert threshold via `COST_ALERT_THRESHOLD_USD`.
- **`GET /health/detailed`**: structured health endpoint for monitoring — reports watcher, scheduler, ingest queue, graph cache, DB, and Qdrant status; never returns 500.
- **`purpose.md` suggestions (K6)**: when ingest detects scope drift, the review queue proposes an addition to the vault's purpose statement; approving it applies the change in place.
- **`schema.md` co-evolution (opt-in, K6)**: recurring frontmatter conventions become schema-rule proposals routed through the review queue.
- **Graph community drill-down**: clicking a community in the graph opens a panel listing its members and a cohesion score; a low-cohesion alert fires when the community is dispersed.
- **Graph edge weight decomposition**: clicking an edge shows the contribution of each of the 4 signals — direct links, shared sources, Adamic-Adar, type affinity.
- **Auto-generated chat titles**: conversations receive a generated title after the first exchange; the conversation list shows a preview of the last message.
- **27 E2E Playwright scenarios** covering the full vault stack.

## Changed

- **Button design system (`.syn-btn*`)**: unified button tokens across the entire UI; 61 non-theme-safe colour usages resolved.
- **Error boundary per section**: a crash in one section no longer produces a blank page — a clear message with a Retry action appears instead.
- **Version display**: the displayed version number is now dynamic from the GitHub release, shown consistently throughout the UI.

## Fixed

- **11 UX regressions from formal audit**: keyboard focus management, dark-mode Review queue rendering, label accuracy, "0 pages" hint text, and ARIA roles corrected.
