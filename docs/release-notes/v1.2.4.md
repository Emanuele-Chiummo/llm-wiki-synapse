# Synapse v1.2.4 — Settings Redesign and Production Frontend Image

_Released: 2026-07-04_

The 3,987-line monolithic settings panel is replaced by a two-level navigation with 16 focused pages; loop-limit controls are now adjustable in-app without environment variable edits; and the frontend ships as a production GHCR image for the first time. Ruff/mypy/eslint/tsc clean.

## Added

- **Settings redesign (ADR-0055)** — the monolithic 3,987-line settings panel is replaced by a two-level navigation with 16 focused pages under `settings/sections/*`; same functionality, much greater clarity [F16].
- **Loop-limit controls in Settings** — `max_iter` and `token_budget` for Deep Research and lint (S14–S18) are now adjustable in-app without environment variables (I7) [F10, K2].
- **Production frontend image** — a multi-stage Vite → nginx build with an integrated API reverse-proxy is published to GHCR on every release; the web client is deployable with a single `docker compose pull` [F15, F16].

## Fixed

- **Type reclassification memory** — pages already examined are now remembered across runs; no more duplicate AI billing on subsequent runs and completion is real [F18, K8].
