Synapse Clipper — Settings

Connection

URL of your Synapse FastAPI service (no trailing slash). Example: http://localhost:8000 or your Tailscale hostname.
The token configured via CLIP_TOKEN in your Synapse environment. Stored encrypted by Chrome. Never shown in full after saving.

Extension Identity

Add chrome-extension://<this-id> to the CLIP_ALLOWED_ORIGINS env var in your Synapse configuration so the server accepts requests from this extension.

Cloudflare Access (optional)

The CF-Access-Client-Id for the service token issued in your Cloudflare Zero Trust dashboard (ends in .access). Only required when your Synapse backend is behind Cloudflare Access. Clear this field and save to disable Cloudflare Access headers.
The CF-Access-Client-Secret value. Stored encrypted by Chrome. Leave blank to keep the currently saved value.
Security reminder: Keep CLIP_TOKEN private — anyone with the token can write files to your Synapse vault. Use a long random string (32+ chars). Rotate via your environment and restart Synapse. The server validates Origin, body size, and path safety independently of this token.
